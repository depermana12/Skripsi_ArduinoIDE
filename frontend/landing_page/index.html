Kebun pintar berbasis IoT

Bertujuan untuk merancang dan mengimplementasikan platform berbasis IoT yang fleksibel untuk pemantauan dan kendali jarak jauh tanaman sayur dari berbagai skala.

Pemantauan dan kendali

Aplikasi teknologi untuk memperoleh data analitik dari lingkungan pertumbuhan tanaman untuk dipelajari sebagai acuan pengambilan keputusan selanjutnya.

Data yang lebih terukur, seperti keadaan tanah, kelembapan, intensitas matahari dapat dipelajari untuk mencapai hasil produksi yang optimal.

Siklus kebun pintar

Observasi, diagnostik, pengambil keputusan dari mikrokontroler, sensor, aktuator dan aplikasi.

  • Mikrokontroler dan sensor

    Akusisi data sensor untuk melacak pertumbuhan lingkungan tanaman sayur

  • Jaringan komunikasi

    Transmisi data sensor nirkabel berbasis internet of things

  • Aplikasi cloud

    Menangani paket data, mengirim dan menjadwalkan data
